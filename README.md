#### To Run this project 

```
git clone https://github.com/Yazilim-Muhendisligi-Uygulamasi/recipes-app.git
cd recipes-app
```
to run the backend(node.js)
```
cd backend
npm install 
npm start
```
to run the frontend(angular)
```
cd frontend
npm install
ng serve --port 8081
```
### NOTE:For the project to work, mongoDB must be installed on your computer.
